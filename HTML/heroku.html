HEROKU

Step 1

Gå til Heroku.com og login/sign in

Step 2

Gå inn på new og trykk på "Create new app"

Step 3

Skriv inn App name, choose a region og eventuelt "add pipeline" også klikk "create app"

Step 4

Gå inn på "deploy" og trykk på github hvis du vill linke github reposetoren din eller gå på heroku CLI kan du følge steppene får å manuelt koble github coden til heroku

how to deploy code to heroku via heroku CLI

Step 1

Først så går du på commands og cd til din git coden din.

Step 2

Så skriv inn "heroku login", "git add ." også "git commit -am 'make it better'" får å pushe det du har gjort til github

Step 3

Etter det skriver du in "heroku git:remote -a reposetoryname" får å linke dem

Step 4

Så skriv inn inn git add og git commit. Etter det skriv inn "git push heroku master" eller "git push heroku main" hvis det er din branch navn
