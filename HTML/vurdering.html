VURDERINGER

Fordeler og ulemper med Github

Fordeler

GitHub er en kodevertsplattform for versjonskontroll og samarbeid. Den lar deg og andre jobbe sammen om prosjekter fra hvor som helst. Githubs pages er en lett å fin måte å få web coden på en websiden ute på internettet.

Ulemper

Github pages er ikke så fleksiblet når det gjelder kode og er ikke kompatibel med node apps. Github pages er ikke en webside som er anbefalt får profisjonelle websider heller.

Fordeler og ulemper med Heroku

Fordeler

Heroku er en litt kraftigere werktøy en github pages. Den kan hoste nesten hva som helst på nesten vilken som hels type code fra app til webside.

Ulemper

Ulempen med dette er det at heroku er veldig avansert å lære og hvis du lager en node app så blir det mye å fårholde seg til hvis dette er din første gang
